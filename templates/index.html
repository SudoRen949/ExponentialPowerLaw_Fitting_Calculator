1. Theoretical Mathematical Discussion

In scientific analysis, data patterns often follow non-linear scaling laws rather than linear trends. Instead of modifying the traditional least-squares equations to manage non-linear trends directly (which introduces computationally demanding, multi-variable polynomial operations), we leverage **linear transformations** using natural logarithms.

Model A: Exponential Fitting $$$$ $y = ae^{bx}$ $$$$

Applying the natural logarithm transforms the relation into a linear format:

$$\ln(y) = \ln(a) + bx$$

By mapping variables where $$$$ $Y = \ln(y), A_0 = \ln(a), A_1 = b$ $$$$ it converts into standard linear regression: $$$$ $Y = A_0 + A_1x$.

Model B: Power-Law Fitting $$$$ $y = ax^b$ $$$$

Applying natural logarithms to both sides yields a fully linearized system:

$$\ln(y) = \ln(a) + b\ln(x)$$

By setting $$$$ $Y = \ln(y), X = \ln(x), A_0 = \ln(a)$ $$$$ we get a clean linear baseline: $$$$ $Y = A_0 + bX$.
